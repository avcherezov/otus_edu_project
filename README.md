## Тестовый проект по курсу "Архитектура и шаблоны проектирования"

### Описание
За основу взяты несколько рабочих наработок по автоматизации тестирования через библиотеку Selenium, попробовал переделать их согласно полученным знаниям на курсе.

В проекте отражены следующие практики из курса:
- обработка исключений, добавил класс ExceptionHandler и теперь могу легко расширять массив известных ошибок в файле errors.py
- добавил абстракции IObject, ICommand
- паттерн Singleton, файл driver.py класс UObject
- паттерн Команда, в папке pages для описания страниц используется паттерн Page Object, добавил к нему паттерн Команда и в классе Command происходит управление
- паттерн Адаптер, методы для взаимодействия с элементами сайта ничего не знают о используемой библиотеке, что позволяет в случае необходимости внести изменения только в файле base.py классе BasePage
- паттерн Декоратор, в файле common.py функция func_info используется как декоратор, добавляя объектам новую функциональность
